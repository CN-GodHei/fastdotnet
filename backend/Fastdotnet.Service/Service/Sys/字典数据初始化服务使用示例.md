# 字典数据初始化服务使用示例

## 服务说明

已将字典数据初始化的核心逻辑抽取到 `IFdDictInitializerService` 服务中，可以在任何地方复用。

## 文件位置

- **接口**: `Fastdotnet.Service.IService.Sys.IFdDictInitializerService`
- **实现**: `Fastdotnet.Service.Service.Sys.FdDictInitializerService`

## 使用方法

### 1. 在构造函数中注入服务

```csharp
public class MyService
{
    private readonly IFdDictInitializerService _dictInitializerService;

    public MyService(IFdDictInitializerService dictInitializerService)
    {
        _dictInitializerService = dictInitializerService;
    }

    public async Task InitializeCustomDictDataAsync()
    {
        // 准备数据
        var dictTypeAndDataList = new List<DictTypeAndData>
        {
            new DictTypeAndData
            {
                fdDictType = new FdDictType
                {
                    Id = "自定义类型 ID",
                    Name = "自定义类型名称",
                    Code = "CUSTOM_01",  // 可选，不填会自动生成
                    SysFlag = YesNoEnum.N,
                    OrderNo = 100,
                    Remark = "备注",
                    Status = StatusEnum.Enable
                },
                fdDictData = new List<FdDictData>
                {
                    new FdDictData
                    {
                        Id = "数据 ID 1",
                        DictTypeId = "自定义类型 ID",
                        Code = "CUSTOM_01_001",  // 可选，不填会自动生成
                        Label = "选项 1",
                        Value = "value1",
                        ValueType = DictValueType.String,
                        OrderNo = 1,
                        Remark = "备注",
                        Status = StatusEnum.Enable
                    },
                    new FdDictData
                    {
                        Id = "数据 ID 2",
                        DictTypeId = "自定义类型 ID",
                        Label = "选项 2",
                        Value = "value2",
                        ValueType = DictValueType.String,
                        OrderNo = 2,
                        Remark = "备注",
                        Status = StatusEnum.Enable
                    }
                }
            }
        };

        // 调用服务保存数据（幂等操作）
        await _dictInitializerService.SaveDictDataAsync(dictTypeAndDataList);
    }
}
```

### 2. 在控制器中使用

```csharp
[ApiController]
[Route("api/[controller]")]
public class DictController : ControllerBase
{
    private readonly IFdDictInitializerService _dictInitializerService;

    public DictController(IFdDictInitializerService dictInitializerService)
    {
        _dictInitializerService = dictInitializerService;
    }

    [HttpPost("import")]
    public async Task<IActionResult> ImportDictData([FromBody] List<DictTypeAndData> dictTypeAndDataList)
    {
        try
        {
            await _dictInitializerService.SaveDictDataAsync(dictTypeAndDataList);
            return Ok(new { message = "导入成功" });
        }
        catch (Exception ex)
        {
            return BadRequest(new { message = $"导入失败：{ex.Message}" });
        }
    }
}
```

## 核心特性

### ✅ 幂等性
- 多次调用不会产生重复数据
- 只插入数据库中不存在的项

### ✅ 自动生成 Code
- **字典类型 Code**: 如果未填写，会从 Name 中提取或自动生成（格式：`CODE_01`, `CODE_02`...）
- **字典数据 Code**: 如果未填写，会自动生成（格式：`CODE_01_001`, `CODE_01_002`...）

### ✅ 智能续号
- 自动计算每个类型下已有的最大 Code 序号
- 从最大值的下一个数字开始生成

### ✅ 批量处理
- 一次性获取所有现有数据进行对比
- 使用 `InsertRangeAsync` 批量插入，性能优化

## 数据格式说明

### DictTypeAndData 结构

```csharp
public class DictTypeAndData
{
    /// <summary>
    /// 字典类型
    /// </summary>
    public FdDictType fdDictType { get; set; }
    
    /// <summary>
    /// 字典数据列表
    /// </summary>
    public List<FdDictData> fdDictData { get; set; }
}
```

### FdDictType 必填字段

```csharp
new FdDictType
{
    Id = "唯一 ID",           // 必填
    Name = "类型名称",        // 必填
    // Code 可选，会自动生成
    SysFlag = YesNoEnum.Y,   // 是否系统内置
    OrderNo = 100,           // 排序号
    Status = StatusEnum.Enable  // 状态
}
```

### FdDictData 必填字段

```csharp
new FdDictData
{
    Id = "唯一 ID",              // 必填
    DictTypeId = "父类型 ID",    // 必填，关联 fdDictType.Id
    // Code 可选，会自动生成
    Label = "显示标签",          // 必填
    Value = "实际值",            // 必填
    ValueType = DictValueType.String,  // 值类型
    OrderNo = 1,                 // 排序号
    Status = StatusEnum.Enable   // 状态
}
```

## 应用场景

### 1️⃣ 插件初始化
```csharp
// 插件启动时注册自己的字典数据
public class MyPlugin : IPlugin
{
    private readonly IFdDictInitializerService _dictInitializerService;

    public async Task InitializeAsync()
    {
        var pluginDictData = new List<DictTypeAndData> { /* ... */ };
        await _dictInitializerService.SaveDictDataAsync(pluginDictData);
    }
}
```

### 2️⃣ 后台管理界面
```csharp
// 管理员通过界面添加字典类型和数据
[HttpPost]
public async Task<ActionResult> CreateDictType(DictTypeCreateDto dto)
{
    var dictTypeAndData = MapToDictTypeAndData(dto);
    await _dictInitializerService.SaveDictDataAsync(new List<DictTypeAndData> { dictTypeAndData });
    return Ok();
}
```

### 3️⃣ 数据迁移/导入
```csharp
// 从旧系统迁移字典数据
public async Task MigrateLegacyDataAsync()
{
    var legacyData = await LoadFromLegacySystem();
    var dictTypeAndDataList = ConvertToDictTypeAndData(legacyData);
    await _dictInitializerService.SaveDictDataAsync(dictTypeAndDataList);
}
```

## 注意事项

1. **Id 必须唯一** - 确保每个字典类型和数据的 Id 在系统中唯一
2. **DictTypeId 关联** - FdDictData.DictTypeId 必须与对应的 FdDictType.Id 一致
3. **Code 格式化** - 
   - 类型 Code: `CODE_XX` (2 位数字)
   - 数据 Code: `CODE_XX_XXX` (3 位数字)
4. **事务支持** - 如需原子操作，可在外部包装事务

## 性能优势

- ✅ **批量查询** - 一次性获取所有现有数据
- ✅ **内存对比** - 在内存中进行数据对比，避免频繁数据库查询
- ✅ **批量插入** - 使用 `InsertRangeAsync` 而非循环单条插入
- ✅ **预处理优化** - 预先分组和计算最大序号，避免嵌套循环
